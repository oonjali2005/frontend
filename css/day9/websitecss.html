CSS Complete Guide

Learn All Important CSS Concepts

What is CSS?

CSS (Cascading Style Sheets) is used to style and design HTML webpages. It controls colors, layouts, spacing, animations and responsiveness.

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

CSS Selectors

Selectors are used to target HTML elements. Example: Element Selector, Class Selector, ID Selector, Universal Selector and Group Selector.

Colors and Backgrounds

CSS provides color names, HEX values, RGB values and gradients to style webpages.

Box Model

Every HTML element contains Content, Padding, Border and Margin.

Margin and Padding

Margin creates space outside the element. Padding creates space inside the element.

Border and Border Radius

Border adds outline around elements. Border-radius creates rounded corners.

Display Property

Display controls how elements appear. Common values are block, inline, inline-block and none.

Position Property

Position is used to place elements. Types: Static, Relative, Absolute, Fixed and Sticky.

Flexbox

Flexbox is a one-dimensional layout system. It helps align items horizontally or vertically.

Grid Layout

CSS Grid is a two-dimensional layout system. It manages rows and columns efficiently.

Transform Property

Transform changes the appearance of elements. Example: scale(), rotate(), translate().

Transition Property

Transition creates smooth changes between states.

Animation

CSS Animation adds movement and visual effects using keyframes.

Pseudo Classes

Pseudo classes define special states. Examples: :hover, :focus, :active.

Pseudo Elements

Pseudo elements style specific parts of elements. Examples: ::before and ::after.

Media Queries

Media Queries make websites responsive for mobiles, tablets and desktops.

Thank You

CSS helps create attractive and responsive websites.
