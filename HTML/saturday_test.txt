1. What is HTML and why is it called a markup language?

i) HTML stands for HyperText Markup Language.
It is used to create and structure web pages.

ii) HTML is called a markup language because it uses tags to mark and organize content like headings, paragraphs, images, and links etc..

example:
        <h1>Welcome</h1>
        <p>hello world</p>

2. What is the difference between HTML and HTML5?

HTML:
     HTML is the basic markup language used to create web pages
     Older version and also allow Limited multimedia support
     

HTML5:
     whereas HTML5 is the latest version of HTML with advanced features. 
     HTML5 supports multimedia elements like audio and video without external plugins and introduces semantic tags such as `<header>`, `<footer>`, and `<section>`. 
     It also provides better performance, local storage support, and improved compatibility with modern websites and mobile devices.

8.. Why do we use the <source> tag inside audio and video tags?

             The <source> tag is used to specify multiple media files for audio and video.
             It helps browsers choose supported file formats.

Example: 
        <video controls>
          <source src="movie.mp4" type="video/mp4">
          <source src="movie.ogg" type="video/ogg">
       </video>
