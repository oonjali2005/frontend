1. WHAT IS HTML?

HTML stands for HyperText Markup Language. HTML is used to create webpages and display content in browsers.

HTML is called a markup language because it uses tags to structure the webpage content.

HTML is not a programming language. It is a markup language used for designing webpage structure.

2. HISTORY OF HTML

HTML was created by Tim Berners-Lee in 1991.

HTML5 is the latest version of HTML.

HTML5 introduced semantic tags, audio, video, local storage and many modern features.

3. BASIC STRUCTURE OF HTML

Basic HTML Structure:


<!DOCTYPE html>
<html>

<head>
    <title>My Webpage</title>
</head>

<body>

    Content goes here

</body>

</html>

DOCTYPE defines the HTML version.

head contains title and meta information.

body contains visible webpage content.

4. HTML TAGS

HTML tags are enclosed inside angle brackets.

Examples:

  • <h1>
  • <p>
  • <a>
  • <img>
  • <table>
  • <form>

Types of Tags

  1. Container Tags
  2. Empty Tags

Example of Container Tag: <p>Hello</p>

Example of Empty Tag: <br>

5. HTML ELEMENTS

HTML element includes opening tag, content and closing tag. 


<p>This is paragraph</p>

Everything from opening tag to closing tag is called an HTML element.

6. HTML ATTRIBUTES

Attributes provide extra information about elements.

Syntax:


<tagname attribute="value">

Examples:


<a href="https://www.google.com">Google</a>

<img src="image.jpg" width="200">

href, src and width are attributes.

14. BLOCK ELEMENTS

Block elements take full width.

  • <div>
  • <p>
  • <h1>
  • <section>
  • <table>

15. INLINE ELEMENTS

Inline elements take only required width.

  • <span>
  • <a>
  • <b>
  • <i>
  • <img>

16. SEMANTIC TAGS

Semantic tags clearly describe content meaning.

  • <header>
  • <nav>
  • <section>
  • <article>
  • <footer>

17. NON SEMANTIC TAGS

Non semantic tags do not describe content meaning.

  • <div>
  • <span>
